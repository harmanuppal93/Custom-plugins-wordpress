== Description ==

Advanced CF7 DB – Reply Back is an add-on of Advanced Cf7 DB, currently the Users(administrator) find it hard to reply back to the users that have been contacted them and there is no instance in the Advanced CF7 DB. This plugin fulfills the feature to reply back to the users. It is simple to use and no extra hardwork required to rely back to the users. Just a click, enter your data and submit, your user will be replied by you. Its that easy and simlpe to use. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

= Install via WordPress Admin =
1. Get Ready the zip file of the plugin.
2. Go to Admin → Plugins → Add New.
3. Click on "Upload Plugin".
4. Get upload the plugin zip files here and click on "Install now".
5. You can see "Advanced CF7 DB – Reply Back" plugin in plugin list now and Activate it.

= Install via FTP =
1. You need to unzip plugin files.
2. In FTP, Go to wp-content/plugins directory.
3. Get Upload the plugin here.
4. You can see "Advanced CF7 DB - Reply Back" plugin in plugin list now and Activate it.
